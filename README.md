# NIST RMF AI

> By [MEOK AI Labs](https://meok.ai) — NIST AI Risk Management Framework (AI RMF 1.0) compliance assessment

## Installation

```bash
pip install nist-rmf-ai-mcp
```

## Usage

```bash
python server.py
```

## Tools

### `assess_ai_system`
Assess an AI system against all four NIST AI RMF core functions (GOVERN, MAP, MEASURE, MANAGE).

### `trustworthy_ai_check`
Evaluate AI system against NIST trustworthy AI characteristics.

### `map_impacts`
Map potential impacts of an AI system across stakeholder groups.

### `risk_controls`
Recommend risk controls aligned to NIST AI RMF subcategories.

### `crosswalk_regulations`
Crosswalk NIST AI RMF requirements to other frameworks (EU AI Act, ISO 42001).

### `generative_ai_profile`
Assess against NIST AI 600-1 Generative AI Profile (July 2024).
